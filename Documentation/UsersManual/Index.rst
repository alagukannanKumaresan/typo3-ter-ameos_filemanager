Users manual
============

.. only:: html

	This chapter describes how to use the extension from a user point of view.


There are two types of rights in this extension : Read and write. According to his rights a frontend user will be able to access different functions. 
In this section we will see what can be done by **FE Users**. For a detailed section on user rights, please see the  :ref:`Administrator manual <adminmanual>`
